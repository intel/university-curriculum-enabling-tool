# User Guides

University Curriculum Enabling Tool supports three distinct personas, each tailored to a specific academic role. Select your persona below to get started.

## Personas

### Curriculum Builder

Designed for faculty members responsible for institutional curriculum design.

**Key capabilities:**

- Add and manage AI language models
- Create and manage courses and programmes
- Export an installation package for Expert Advisors and Learning Companions

[Curriculum Builder Guide →](./user-guides/curriculum-builder.md)

---

### Expert Advisor

Designed for lecturers who deliver courses using a pre-configured installation package.

**Key capabilities:**

- Load courses from an installation package
- Create presentation slides, FAQs, assessments, and quizzes
- Chat with an AI model, optionally using RAG documents as context

[Expert Advisor Guide →](./user-guides/expert-advisor.md)

---

### Learning Companion

Designed for students engaging with course content and personal study materials.

**Key capabilities:**

- Access available courses
- Generate summaries, FAQs, and quizzes from uploaded documents
- Chat with an AI model, optionally using RAG documents as context

[Learning Companion Guide →](./user-guides/learning-companion.md)
